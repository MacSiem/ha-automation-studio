# Contributing to ha-automation-studio

Thank you for your interest in contributing! This project welcomes contributions from the Home Assistant community.

## How to contribute

1. **Fork** the repository
2. **Create a branch** for your feature or fix: `git checkout -b feature/my-feature`
3. **Make your changes** and write tests where applicable
4. **Open a Pull Request** with a clear description of what you changed and why

## Areas where help is needed

- Improving YAML analysis accuracy and coverage
- Home Assistant API integration (entity discovery)
- Adding support for additional AI models (OpenAI, Ollama/local LLMs)
- CLI improvements and new commands
- Documentation, examples, and tutorials

## Before contributing

Please open an issue first to describe your proposal. This avoids duplicate work and ensures the change aligns with project goals.

## Code style

- Python 3.9+
- Follow PEP 8
- Add docstrings to all public functions
- Write tests for new functionality

## Questions?

Open an issue with the `question` label.
